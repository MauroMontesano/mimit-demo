Breadcrumbs and Navbar
Since the "navbar" section is also used as the navigation component for these docs, this section will cover informations for both breadcrumbs and navbar
The navbar will automatically build the menu according to the routing of your application (defined as you can see here). If you want to remove some values you can make a copy of the "this.route.config" value and edit it, passing it with the [config] attribute
The breadcrumbs will use the routing too, building its path from the current route. Unlike the navbar, you can't edit the breadcrumbs behavior
Values inside data are used both by the navbar and the breadcrumb.
  • breadcrumbs: name of the route
  • icon: icon in the menu (fontawesome class)
  • disableBreadcrumbLink: Wheter or not a certain part of the breadcrumbs should be clickable
