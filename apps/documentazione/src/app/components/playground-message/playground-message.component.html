# Message service Used to show messages to the user. All messages last for 10s. The error message must be closed by the user to be dismissed. The **show()** method of the EafMessageService is used, receiving a Message instance as parameter. Alternatively the methods **info()**, **success()**, **warning()**, **error()** can be used by passing a content and an optional title. Based upon the **Message** model: ------------------------------------------------------------------------- | Property | Type | Description | |----------------------|-------|----------------------------------------| | [text] | string | Message content. | | [title] | string | Message title. | | [type] | MessageType | Used to determine the message type. By default it is an INFO. | | [i18nPlaceHolder] | any | Values for i18n placeholder. |
