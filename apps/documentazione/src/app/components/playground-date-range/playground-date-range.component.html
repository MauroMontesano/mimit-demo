# Date range This element is used in a form as date range input. ------------------------------------------------------------------------- | [Input] / (Output) | Type | Description | |----------------------|-------|----------------------------------------| | [label] | i18n Key | this will be used for the label (and also as base for the id of the internal input). | | [fromLabel] | string | Label to be used for the 'from' input. | | [toLabel] | string | Label to be used for the 'to' input. | | [ext-id] | string without internal spaces | For the cases where label itsn't unique you have to define this input to guarantee the uniqueness of the internal input id. | | [formControl] | FormControl | Angular Form control to map with this input. | | [size] | BoostrapSize | Size of form group. String with format 'xs\|sm\|md\|lg' like '12\|12\|6\|3'. Default '12\|12\|6\|4\'. | | [error-labels] | ErrorLabel[] | This is used to display errors linked with this form element. | | [placeholder] | i18n Key | This is the default input text. | | [tooltip] | TooltipModel | Shows a tooltip that explains the input. | | [readonly-input] | boolean | If true the input is readonly. Note: in this case the placeholder is hidden. | | [disable-input] | boolean | If true the input is disable-input. Note: in this case the placeholder is hidden. | | [warning] | string | A warning message that will always display bottom the input. | | [regex] | Regexp or string or string[] | Regular expression. | | [minDate] | string \| Date \| Moment | Min date. | | [maxDate] | string \| Date \| Moment | Max date. | | [disableFrom] | boolean | disable only from field | | [disableFrom] | boolean | disable only from field |
