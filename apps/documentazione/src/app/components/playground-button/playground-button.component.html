# Button This element is used as button. ------------------------------------------------------------------------- | [Input] / (Output) | Type | Description | |----------------------|-------|----------------------------------------| | [title] | string | Defines title property. | | [color] | string | For selecting color (look) of button. One of 'default', 'primary', 'success', 'warning', 'danger'. | | [ext-id] | string without internal spaces | For the cases where label itsn't unique you have to define this input. | | [icon] | string | Defines fa-icon to be shown at the left of the button. | | [iconRight] | string | Defines fa-icon to be shown at the right of the button. | | [onlyIcon] | boolean | Defines fa-icon to be shown alone without text | | [type] | string | Defines type of button | | [extraClass] | string | Additional classes to be added to the button. | | [isLink] | boolean | Whether the button should appear as a link. | | [disabled] | boolean | Disables the button. | | (click) | EventEmitter | Emits button click event. |
