# Panel xpand This element is a panel with optional accordion behavior, actions, and inside panel template. It is used both for both accordions and expandable panels. If you need an accordion, i.e. only one panel can be opened at a time, wrap the component within an "AccordionManager". ------------------------------------------------------------------------- | [Input] / (Output) | Type | Description | |----------------------|-------|----------------------------------------| | [panelTitle] | i18n Key or String | this will be use to define title. Can be a string or a i18n key | | [i18n] | i18n placeholder map | it's a map key value to replace some placeholder in i18n message | | [panelContent] | template | Template used to fill the space beetween title and actions. It will share a model with the template. | | [accordion] | template | Template used for the accordion. It will share a model with the template. | | [actions] | ActionsItem [] | An array of action items used in the actions section. | | [model] | any | The model that the panel will share with templates section. | | [isOpen] | boolean | Whether the accordion will be initially open. Default false. | | [editMode] | number or Function | Passed to have the button "edit" in the accordion (used only in wizards recap). If number it represent the queryParam path. If function it takes the id of the step | | (toggle) | emitter(boolean) | Emits events when the panel is toggled: true for open, false for closed. | | [elementSizes] | {{'{'}} title?: string, extra?: string, actions?: string } | Set the sizes of panel elements (pass bootstrap pipe string). | > You can also use class="transparent" on eaf-accordion-manager to made it transparent
