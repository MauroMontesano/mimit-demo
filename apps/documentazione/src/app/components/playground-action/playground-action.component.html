# Action Represents an action. Based on the **ActionItem** model. ------------------------------------------------------------------------- | [Input]/[Output] | Type | Description | |----------------------|-------|----------------------------------------| | [action] | ActionItem | The action to execute. | | [model] | BaseModel | The item on which to execute the action. | | [onlyIcon] | boolean | Whether to show only the icon and not the action name. | | [disabled] | boolean | Whether the action should be disabled. | | [dynamicTitle] | string | Overrides the actionItem name. | | [extraClass] | string | Extra classes to add. | | [loading] | boolean | if true change icon in spinner and disable the action |
