# Textarea This element is used in a form as textarea input. ------------------------------------------------------------------------- | [Input] / (Output) | Type | Description | |----------------------|-------|----------------------------------------| | [label] | i18n Key | this will be used for the label (and also as base for the id of the internal input). | | [ext-id] | string without internal spaces | For the cases where label itsn't unique you have to define this input to guarantee the uniqueness of the internal input id. | | [formControl] | FormControl | Angular Form control to map with this input. | | [size] | BoostrapSize | Size of form group. String with format 'xs\|sm\|md\|lg' like '12\|12\|6\|3'. Default '12\|12\|6\|4\'. | | [size-label] | BoostrapSize | Size of label. String with format 'xs\|sm\|md\|lg' like '12\|12\|6\|3'. Default '12\|12\|12\|12\'. | | [size-input] | BoostrapSize | Size of input. String with format 'xs\|sm\|md\|lg' like '12\|12\|6\|3'. Default '12\|12\|12\|12\'. | | [size-messages] | BoostrapSize | Size of message container. String with format 'xs\|sm\|md\|lg' like '12\|12\|6\|3'. Default '12\|12\|12\|12\'. | | [error-labels] | ErrorLabel[] | This is used to display errors linked with this form element. | | [placeholder] | i18n Key | This is the default input text. | | [tooltip] | TooltipModel | Shows a tooltip that explains the input. | | [readonly-input] | boolean | If true the input is readonly. Note: in this case the placeholder is hidden. | | [disable-input] | boolean | If true the input is disabled. Note: in this case the placeholder is hidden. | | [actions] | ActionItem[] | Actions displayed on the right of input. | | [warning] | string | A warning message that will always display bottom the input. | | [rows] | number | Number of textarea rows. | | [maxlength] | number | Limits the number of characters that can be inserted. |
